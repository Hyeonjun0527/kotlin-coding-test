# BFS 냄새 찾기 연습

> 문제 설명만 보고 "이거 BFS다"를 떠올릴 수 있는가?

---

## 1. 점프 점프 (BOJ 11060)

### <문제>
1×N 미로의 각 칸에 정수가 쓰여 있다.
i번째 칸에 A_i가 쓰여 있으면, 그 칸에서 오른쪽으로 1칸~A_i칸 중 하나로 점프할 수 있다.
가장 왼쪽에서 출발해 가장 오른쪽 끝까지 가는 최소 점프 횟수를 구하라.
갈 수 없으면 -1을 출력한다.

### <정답>
**냄새:** `가장 왼쪽 → 가장 오른쪽` + `최소 점프 횟수` → **"시작점에서 도착점까지 최소 횟수"**

한 번 점프 = 비용 1이다. 칸마다 갈 수 있는 다음 칸이 여러 개(1~A_i칸 뒤)이고, 매 점프의 비용이 동일하게 1이다.
"비용이 전부 1인 그래프에서 최단거리"는 BFS의 전형적인 패턴이다.
1번 칸을 큐에 넣고, 갈 수 있는 칸들을 다음 레벨로 확장해 나가면, N번 칸에 처음 도달한 시점의 레벨이 곧 최소 점프 횟수다.
**"한 번 행동 = 비용 1, 최소 횟수 = BFS"**

## 2. 경쟁적 전염 (BOJ 18405)

### <문제>
N×N 시험관에 1번~K번 바이러스가 있다.
매 초마다 모든 바이러스가 상하좌우로 동시에 증식한다.
단, 번호가 낮은 바이러스부터 먼저 증식하며, 이미 바이러스가 있는 칸에는 들어갈 수 없다.
S초 후 (X, Y) 위치의 바이러스 번호를 구하라. 없으면 0.

### <정답>
**냄새:** `매 초마다 상하좌우로 동시에 증식` + `번호가 낮은 것부터 먼저` → **"다중 시작점에서 동시에 한 칸씩 퍼져나감"**

"동시에 퍼져나간다"는 것은 BFS의 레벨 단위 확장과 정확히 같다.
바이러스가 여러 개이므로 **다중 시작점 BFS**다. 큐에 모든 바이러스를 초기 위치로 넣되, 번호가 낮은 것을 먼저 넣으면(정렬 후 삽입) 자연스럽게 "번호 낮은 것 우선" 조건이 충족된다.
한 레벨 = 1초이므로, S레벨까지만 확장하면 된다.
**"동시에 퍼져나감 + 한 번 확장 = 1초 = 다중 시작점 BFS"**

## 3. 전쟁 - 전투 (BOJ 1303)

### <문제>
N×M 격자에 아군(W)과 적군(B)이 배치되어 있다.
같은 팀이 상하좌우로 인접하면 하나의 뭉치이고, N명이 뭉쳐있으면 N²의 위력을 낸다.
아군의 총 위력과 적군의 총 위력을 각각 구하라.

### <정답>
**냄새:** `같은 팀이 상하좌우로 인접 → 하나의 뭉치` + `뭉치 크기의 제곱` → **"연결된 덩어리의 크기를 구하라"**

이 문제는 DFS로도 풀 수 있다. 하지만 BFS로도 자연스럽다.
한 칸에서 출발해 같은 팀인 인접 칸을 큐에 넣으며 확장하면, 큐가 빌 때까지 방문한 칸 수 = 뭉치 크기다.
각 뭉치 크기를 제곱해서 합산하면 총 위력이 나온다.
핵심은 "연결 요소(connected component)의 크기를 구하는 것"이며, BFS든 DFS든 탐색으로 풀린다.
**"연결된 덩어리 크기 구하기 = BFS (또는 DFS)"**

## 정리: BFS 냄새 체크리스트

| 냄새 | 핵심 질문 |
|------|----------|
| 최소 횟수/최단 거리 | "한 번 행동 = 비용 1, 최소 몇 번?" |
| 동시에 퍼져나감 | "매 턴마다 사방으로 동시 확장?" |
| 다중 시작점 | "여러 곳에서 동시에 출발?" |
| 레벨 단위 처리 | "몇 초/몇 단계 후의 상태?" |
| **공통점** | **비용이 균일(=1)하고 최소/동시 확장** |
