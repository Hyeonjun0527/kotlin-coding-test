# 네트워크 스키마 모델링

코드의 `[S*]`, `[V*]` 집합과 1:1로 맞춘 버전이다.

```mermaid
graph TB
  S0["S0 초기화"]
  S1["S1 시작점 순회"]
  C0["C0 시작점 순회 완료?"]
  S2["S2 시작점 방문 여부 검사"]
  C1["C1 시작점이 이미 방문됨?"]
  S3["S3 새 네트워크 시작"]
  S4["S4 BFS 확장"]
  C2["C2 큐가 비었는가?"]
  C3["C3 현재-다음이 연결됨?"]
  C4["C4 다음컴퓨터가 이미 방문됨?"]
  S5["S5 컴포넌트 탐색 완료"]
  S_END["S_END 결과 반환"]

  V0["V0 총컴퓨터수"]
  V1["V1 그래프"]
  V2["V2 visited"]
  V3["V3 네트워크수"]
  V4["V4 탐색큐"]
  V5["V5 현재컴퓨터"]
  V6["V6 다음컴퓨터"]

  S0 --> S1
  S1 --> C0
  C0 -- "YES" --> S_END
  C0 -- "NO" --> S2
  S2 --> C1
  C1 -- "YES" --> S1
  C1 -- "NO" --> S3
  S3 --> S4
  S4 --> C2
  C2 -- "YES" --> S5
  S5 --> S1
  C2 -- "NO" --> C3
  C3 -- "NO" --> C2
  C3 -- "YES" --> C4
  C4 -- "YES" --> C2
  C4 -- "NO" --> C2

  S0 --> V0
  S0 --> V1
  S0 --> V2
  S0 --> V3
  S3 --> V2
  S3 --> V3
  S3 --> V4
  S4 --> V1
  S4 --> V2
  S4 --> V4
  S4 --> V5
  S4 --> V6
```

## S/V 싱크 체크 결과
- S 집합: `S0,S1,S2,S3,S4,S5,S_END` 코드/스키마 일치
- V 집합: `V0,V1,V2,V3,V4,V5,V6` 코드/스키마 일치
