# 호석이두마리치킨v2 알고리즘 상태 전이 그래프

코드의 `[S]`, `[V]` 주석과 맞춘 버전이다.

```mermaid
graph TB
  S0["S0 초기화"]
  S1["S1 간선 입력 반영"]
  S2["S2 all pairs 거리 계산"]
  C1["현재 BFS 큐가 비었는가"]
  S4["S4 sequence 기반 인접 노드 확장"]
  S3["S3 최적 조합 탐색"]
  SEND["S_END 결과 반환"]

  S0 --> S1 --> S2 --> C1
  C1 -- "NO" --> S4
  S4 --> C1
  C1 -- "YES" --> S3
  S3 --> SEND

  V0["V0 그래프"]
  V1["V1 dist"]
  V2["V2 큐"]
```

## V 상태
- `V0`: 건물 그래프
- `V1`: 최단거리 테이블
- `V2`: BFS 큐
