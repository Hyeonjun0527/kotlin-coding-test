# 강의실 스키마 모델링

```mermaid
graph TB
  A1["S0 상태 초기화"]
  A2["S1 시작시간 기준 정렬"]
  A3["S2 종료시간 힙 초기화"]
  A4["S3 강의 순회"]
  A5["S4 재사용 가능 강의실이면 poll"]
  A6["S5 현재 강의 종료시간 삽입"]
  A7["S_END 힙 크기 반환"]

  B1["V0 강의개수"]
  B2["V1 정렬강의목록"]
  B3["V2 종료시간힙"]
  B4["V3 강의"]
  B5["V4 시작시간"]
  B6["V5 종료시간"]

  A1 --> A2
  A2 --> A3
  A3 --> A4
  A4 -- "YES" --> A5
  A4 -- "NO" --> A7
  A5 --> A6
  A6 --> A4

  A1 --> B1
  A2 --> B2
  A3 --> B3
  A4 --> B4
  A4 --> B5
  A6 --> B6
```

## S/V 싱크 체크 결과
- S 집합: `S0,S1,S2,S3,S4,S5,S_END` 코드/스키마 일치
- V 집합: `V0,V1,V2,V3,V4,V5` 코드/스키마 일치
