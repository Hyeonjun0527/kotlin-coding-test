# 칠무해 스키마 모델링

```mermaid
graph TB
  A1["S0 상태 초기화"]
  A2["S1 점수 순회"]
  A3["S2 힙 크기가 정답개수 미만인가"]
  A4["S3 점수 삽입"]
  A5["S4 더 작은 점수면 교체"]
  A6["S5 정렬 및 포맷팅"]
  A7["S_END 결과 반환"]

  B1["V0 정답개수"]
  B2["V1 선발힙"]
  B3["V2 점수"]
  B4["V3 정렬결과"]

  A1 --> A2
  A2 --> A3
  A3 -- "YES" --> A4
  A3 -- "NO" --> A5
  A4 --> A2
  A5 --> A2
  A2 --> A6
  A6 --> A7

  A1 --> B1
  A1 --> B2
  A2 --> B3
  A6 --> B4
```

## S/V 싱크 체크 결과
- S 집합: `S0,S1,S2,S3,S4,S5,S_END` 코드/스키마 일치
- V 집합: `V0,V1,V2,V3` 코드/스키마 일치
