# 가운데를말해요 스키마 모델링

```mermaid
graph TB
  A1["S0 상태 초기화"]
  A2["S1 숫자 순회"]
  A3["S2 크기 균형 규칙으로 삽입"]
  A4["S3 top 순서 점검"]
  A5["S4 top swap으로 순서 복구"]
  A6["S5 현재 중간값 기록"]
  A7["S_END 결과 반환"]

  B1["V0 입력개수"]
  B2["V1 작은절반힙"]
  B3["V2 큰절반힙"]
  B4["V3 중간값목록"]
  B5["V4 현재수"]
  B6["V5 작은힙Top"]
  B7["V6 큰힙Top"]

  A1 --> A2
  A2 -- "YES" --> A3
  A2 -- "NO" --> A7
  A3 --> A4
  A4 -- "YES" --> A5
  A4 -- "NO" --> A6
  A5 --> A6
  A6 --> A2

  A1 --> B1
  A1 --> B2
  A1 --> B3
  A1 --> B4
  A2 --> B5
  A5 --> B6
  A5 --> B7
```

## S/V 싱크 체크 결과
- S 집합: `S0,S1,S2,S3,S4,S5,S_END` 코드/스키마 일치
- V 집합: `V0,V1,V2,V3,V4,V5,V6` 코드/스키마 일치
