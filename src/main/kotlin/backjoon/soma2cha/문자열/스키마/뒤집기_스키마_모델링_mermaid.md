# 뒤집기 스키마 모델링

```mermaid
graph TB
  A1["S0 상태 초기화"]
  A2["S1 첫 그룹 반영"]
  A3["S2 문자열 순회"]
  A4["S3 그룹 경계 검사"]
  A5["S4 새 그룹 카운트와 이전문자 갱신"]
  A6["S_END 최소 그룹수 반환"]

  B1["V0 문자열길이"]
  B2["V1 이전문자"]
  B3["V2 영그룹수"]
  B4["V3 일그룹수"]
  B5["V4 현재문자"]

  A1 --> A2
  A2 --> A3
  A3 -- "YES" --> A4
  A3 -- "NO" --> A6
  A4 -- "YES" --> A5
  A4 -- "NO" --> A3
  A5 --> A3

  A1 --> B1
  A1 --> B2
  A1 --> B3
  A1 --> B4
  A3 --> B5
```

## S/V 싱크 체크 결과
- S 집합: `S0,S1,S2,S3,S4,S_END` 코드/스키마 일치
- V 집합: `V0,V1,V2,V3,V4` 코드/스키마 일치
