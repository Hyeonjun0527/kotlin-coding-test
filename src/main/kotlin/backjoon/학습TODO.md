문자열(String): 파싱, 찾기(indexOf), 치환, 정렬, 패턴 만들기

시뮬레이션(Simulation): 규칙대로 한 단계씩 진행(로봇 이동, 게임 턴, 달력, 주사위 등)

브루트포스(완전탐색): 범위가 작아서 전부 돌려도 됨(조건 체크가 핵심)

자료구조: 맵/셋/큐/스택으로 규칙 구현(카운팅, 중복 제거, 상태 관리)

정렬: 조건대로 정렬하고 출력(우선순위가 여러 개)

그리디(가끔): “그렇게 하는 게 맞다” 증명은 쉬운데 구현 실수가 잦음

좌표/격자(Grid): 2D 맵, 방향벡터(dx/dy), 경계 체크

수학(가끔): 식은 쉬운데 예외처리(음수, 나눗셈, 오버플로 등) 때문에 구현이 핵심
