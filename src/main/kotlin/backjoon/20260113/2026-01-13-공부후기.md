## 정수 나눗셈 (올림/반올림/버림)

### 1️⃣ 나눗셈 결과 기준 (n / d를 정수로)

어떻게 계산?
```
버림:   n / d
반올림: (n + d/2) / d
올림:   (n + d - 1) / d
```

예시 (n=7, d=4):
```kotlin
val n = 7
val d = 4

println(n / d)             // 1 (버림)
println((n + d/2) / d)     // 2 (반올림)
println((n + d - 1) / d)   // 2 (올림)
```

| 목적 | 공식 | 예 (n=7, d=4) |
|------|------|---------------|
| 버림 | n / d | 1 |
| 반올림 | (n + d/2) / d | 2 |
| 올림 | (n + d - 1) / d | 2 |

### 2️⃣ 숫자 자체를 d 단위로 처리

10 단위로 반올림하려면?
"d 단위로 정렬"
```kotlin
val n = 127
val d = 10

println((n / d) * d)           // 120 (버림)
println(((n + d/2) / d) * d)   // 130 (반올림)
println(((n + d - 1) / d) * d) // 130 (올림)
```

| 목적 | 공식 | 예 (n=127, d=10) |
|------|------|------------------|
| d 단위 버림 | (n / d) * d | 120 |
| d 단위 반올림 | ((n + d/2) / d) * d | 130 |
| d 단위 올림 | ((n + d - 1) / d) * d | 130 |

## 배열/컬렉션 비우기

어떻게 비워?
"컬렉션은 clear(), 배열은 fill()"

| 타입 | 비우는 법 |
|------|-----------|
| MutableList | list.clear() |
| ArrayDeque | q.clear() |
| MutableSet | set.clear() |
| MutableMap | map.clear() |
| BooleanArray | arr.fill(false) |
| IntArray | arr.fill(0) |
| Array | arr.fill(defaultValue) |
| StringBuilder | sb.clear() 또는 sb.setLength(0) |

핵심 기억 포인트:
- **clear()는 컬렉션(MutableList/Set/Map/Deque) 전용**
- **배열(Array 계열)은 clear()가 없어서 fill()로 덮어씀**
- **StringBuilder는 clear() 또는 setLength(0)**

```kotlin
// 컬렉션
val list = mutableListOf(1, 2, 3)
list.clear()  // []

// 배열
val arr = IntArray(5) { it }
arr.fill(0)  // [0, 0, 0, 0, 0]
```

## BFS에서 이전 노드 추적

previousNode 배열 만들기?
"prev 배열에 부모 저장"
```kotlin
val prev = IntArray(n) { -1 }
val q = ArrayDeque<Int>()

q.add(start)
prev[start] = start  // 시작점은 자기 자신

while (q.isNotEmpty()) {
    val cur = q.removeFirst()
    for (next in graph[cur]) {
        if (prev[next] == -1) {  // 방문 안 한 곳
            prev[next] = cur     // 현재가 부모
            q.add(next)
        }
    }
}
```

경로 되살리기?
"끝에서 시작까지 역추적"
```kotlin
val path = mutableListOf<Int>()
var x = end

while (x != start) {
    path.add(x)
    x = prev[x]  // 부모로 이동
}
path.add(start)
path.reverse()  // 뒤집어서 start → end

println(path.joinToString(" -> "))
```
