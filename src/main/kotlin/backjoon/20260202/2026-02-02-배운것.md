안비어있는 가변 크기 문자 리스트
“행은 몇 줄인지 알지만,
각 줄 길이는 다를 수 있다”
```kotlin
val n = readln().toInt()   // 행 개수만 주어지는 경우
val map = MutableList(n) { readln().toCharArray() }
```

비어있는 가변리스트 만들고 넣기
“처음에 몇 줄인지도 모른다
(또는 입력 흐름에 따라 추가된다)”
```kotlin
val map = mutableListOf<CharArray>()

repeat(K) {   // K는 문제에서 보장
    map.add(readln().toCharArray())
}
```

안비어있는 가변 리스트 만들고 덮어쓰기(아래로 개선 필요)
```kotlin
    val map = MutableList(r) { MutableList(c) { ' ' } }

    repeat(r) { i ->
        map[i] = readln().toMutableList()
    }
```

안비어있는 가변 리스트 만들기
```kotlin
val map = MutableList(r) { readln().toMutableList() }
```

안비어있는 불변 리스트 만들기
```kotlin
val map = Array(r) { readln().toCharArray() }
```

코틀린에서 string은 3가지 리스트로 만들수잇음.
"123".toCharArray()      // CharArray
"123".toList()           // List<Char>
"123".toMutableList()    // MutableList<Char>
