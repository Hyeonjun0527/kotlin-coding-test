## 채워진 가변 크기 문자 리스트를 만드려면?
“행은 몇 줄인지 알지만,
각 줄 길이는 다를 수 있다”
```txt
3
ABCDE
hi
Kotlin!!
```

```kotlin
val n = readln().toInt()   // 행 개수만 주어지는 경우
val map = MutableList(n) { readln().toCharArray() }
```

```txt
map
 ├─ map[0] -> ['A','B','C']
 ├─ map[1] -> ['h','i','!']
 └─ map[2] -> ['K','o','t','l','i','n']
```

## 비어있는 가변리스트 만들고 넣으려면?
“처음에 몇 줄인지도 모른다
(또는 입력 흐름에 따라 추가된다)”

```txt
3
cat
DOGG
12
```

```kotlin
val map = mutableListOf<CharArray>()

repeat(K) {   // K는 문제에서 보장
    map.add(readln().toCharArray())
}
```

```txt
map
 ├─ map[0] -> ['c','a','t']
 ├─ map[1] -> ['D','O','G','G']
 └─ map[2] -> ['1','2']
```
