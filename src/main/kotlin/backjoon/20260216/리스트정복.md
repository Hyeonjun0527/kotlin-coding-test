## 추가 (Add)

add(k),add(i,k), addAll(리스트)

add(i,제거한값)은 i위치에 넣고 기존i에 잇던건 +1 하는 것

```kotlin
list.add(10)                 // 맨 뒤에 추가
list.add(0, 99)              // index에 삽입
list.addAll(listOf(7, 8, 9)) // 여러 개 추가
```

## 수정 (Update / Replace)

list[0]

```kotlin
list[0] = 111        // 인덱스로 수정
```

## 삭제 (Remove)

remove, removeAt, clear
removeFirst, removeLast

```kotlin
list.remove(10)        // 값 10을 "하나" 삭제 (있으면 true)
list.removeAt(0)       // index 위치 삭제 (삭제된 값 반환)
list.removeAll(listOf(1, 2))   // 값들 전부 삭제
list.clear()           // 전부 삭제
list.removeFirst()      // 첫 원소 삭제 (비어있으면 예외)
list.removeLast()       // 마지막 원소 삭제 (비어있으면 예외)
```

### 커스텀 정렬 : 내부 요소로 정렬 

sortedWith(
compareBy {?}
.thenBy {?}
)

```kotlin
data class User(val name: String, val age: Int)

val users = listOf(
  User("Kim", 30),
  User("Lee", 20),
  User("Park", 20)
)

val byAge = users.sortedWith(
    compareBy { it.age }
    .thenBy { it.name })

val byAge2 = users.sortedWith(
    compareByDescending { it.age }
    .thenBy { it.name })

```
