# 2026-02-06 배운 것

## 문자열 연결 함정

"" + "" 반복하면 메모리 초과?
"매번 새 문자열 만들기 때문"
```kotlin
var s = ""
for (i in 0 until n) {
    s = s + "a"  // ❌ 매번 새 문자열 생성
}
```

시간복잡도는?
"현재 s의 길이가 비용"
```
비용: 0 + 1 + 2 + ... + n
=> O(n²)
```

StringBuilder 써야?
"한 번에 모아서 만들기"
```kotlin
val sb = StringBuilder()
for (i in 0 until n) {
    sb.append("a")  // ✅ 기존 버퍼에 추가
}
val s = sb.toString()  // O(n)
```
