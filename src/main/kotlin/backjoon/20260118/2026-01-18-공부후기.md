## any vs all

any - 하나라도 조건 만족?
"있냐 없냐"
```kotlin
// 예시 1: 리스트 안에 짝수가 하나라도 있나?
val nums = listOf(1, 3, 5, 8)
val hasEven = nums.any { it % 2 == 0 }
println(hasEven) // true

// 예시 2: 문자열에 숫자가 하나라도 있나?
val str = "abc3d"
val containsDigit = str.any { it.isDigit() }
println(containsDigit) // true
```

all - 전부 조건 만족?
"다 그런가?"
```kotlin
// 예시 3: 리스트의 모든 숫자가 양수인가?
val nums = listOf(1, 2, 3, 4)
val allPositive = nums.all { it > 0 }
println(allPositive) // true

// 예시 4: 비밀번호가 "문자 전부 소문자냐?" 검사
val pwd = "hello"
val isAllLower = pwd.all { it.isLowerCase() }
println(isAllLower) // true
```

## reverse

reverse는 제자리정렬? 아니면 리턴?
"제자리 정렬 (원본 변경)"
```kotlin
val list = mutableListOf(1, 2, 3)
list.reverse()  // list가 바로 [3, 2, 1]로 변경됨
```
