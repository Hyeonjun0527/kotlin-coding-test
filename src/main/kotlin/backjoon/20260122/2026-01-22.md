## removeFirst vs drop

removeFirst()와 drop(1)의 차이?
"원본 변경 vs 새 리스트"
```kotlin
val list = mutableListOf(1, 2, 3)

// removeFirst(): 원본을 바꾸는 수술
list.removeFirst()  // list가 [2, 3]로 변경됨

// drop(1): 복사본 하나 만들어서 보여주기
val newList = list.drop(1)  // 원본 유지, newList = [2, 3]
```
