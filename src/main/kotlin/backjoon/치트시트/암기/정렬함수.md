| 방식                                  | 원본 반영 여부 | 이유                     | 주용도                 | 적용 대상                                        |
| ----------------------------------- | -------- | ---------------------- | ------------------- | -------------------------------------------- |
| `원본 자료구조.sort(l, r+1)`              | 반영됨      | 원본 자체를 범위 정렬           | 원본 일부 직접 정렬         | `IntArray`, `Array<Int>`                     |
| `원본 리스트.subList(l, r+1).sort()`     | 반영됨      | `subList`는 원본 구간 뷰     | `MutableList` 부분 정렬 | `MutableList<Int>`                           |
| `원본 배열.sliceArray(l..r)` 후 `sort()` | 반영 안 됨   | 복사본 배열을 정렬             | 원본 보존하며 계산          | `IntArray`, `Array<Int>`                     |
| `원본 자료구조.slice(l..r).sorted()`      | 반영 안 됨   | 새 리스트 생성               | 잘라서 정렬한 결과만 필요      | `IntArray`, `Array<Int>`, `MutableList<Int>` |
| 정리                                  | 거의 맞음    | 정렬만 할 거면 복사 방식은 보통 불필요 | 실전에서는 직접 범위 정렬 선호   | 상황별 선택                                       |
