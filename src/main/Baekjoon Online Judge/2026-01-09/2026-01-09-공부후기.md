## **1) 출력할 때 joinToString 하면서 소수점 3자리 반올림하기**

```kotlin
val nums = listOf(1.23456, 7.89123)

val out = nums.joinToString(" ") { "%.3f".format(it) }
println(out) // 1.235 7.891
```

---

## **2) 출력할 때 소수점 3자리 반올림하기**

```kotlin
val x = 1.23456
println("%.3f".format(x)) // 1.235
```

---

## **3) 값 자체를 소수점 3자리로 반올림하여 숫자로 만들기**

```kotlin
import kotlin.math.round

val x = 1.23456
val rounded = round(x * 1000) / 1000
println(rounded) // 1.235
```

---

## **4) 리스트 요소를 진짜 7개만 남기고 나머지 제거하기 (원본 변경)**

```kotlin
val list = mutableListOf(1,2,3,4,5,6,7,8,9)

while (list.size > 7) list.removeLast()

println(list) // [1,2,3,4,5,6,7]
```

---

## **5) 리스트에서 마지막 요소 꺼내기 (pop)**

```kotlin
val list = mutableListOf(1, 2, 3)
val last = list.removeLast() // 꺼내면서 제거
println(last) // 3
```

---

## **6) 리스트 요소 제거는 안 하고 앞의 7개만 가져오기 (원본 유지)**

```kotlin
val list = listOf(1,2,3,4,5,6,7,8,9)

val picked = list.take(7)
println(picked) // [1,2,3,4,5,6,7]
println(list)   // 원본 그대로
```

---

## **7) 리스트를 내림차순으로 정렬하기 (원본 변경)**

```kotlin
val nums = mutableListOf(5, 1, 9, 3)
nums.sortDescending()
println(nums) // [9,5,3,1]
```

---

## **8) 리스트를 key 기준(예: 길이)으로 내림차순 정렬하기**

```kotlin
val words = mutableListOf("apple", "kiwi", "banana")

words.sortByDescending { it.length }
println(words) // [banana, apple, kiwi]
```

---

## **9) PriorityQueue를 “내림차순(큰 값 먼저)”으로 만들어 poll()로 순회하기**

```kotlin
val pq = PriorityQueue(compareByDescending<Int> { it })

pq.addAll(listOf(10, 3, 7, 1))

while (pq.isNotEmpty()) {
    println(pq.poll())
}
// 10
// 7
// 3
// 1
```

---

## **10) PriorityQueue를 “정렬된 순서대로 순회”하되, 내부 요소는 그대로 두기**

```kotlin
val pq = PriorityQueue(compareByDescending<Int> { it })
pq.addAll(listOf(10, 3, 7, 1))

val copy = PriorityQueue(pq) // 복사본 만들기

while (copy.isNotEmpty()) {
    println(copy.poll())
}
// 10
// 7
// 3
// 1

println(pq) // 원본 pq는 그대로 유지됨
```
