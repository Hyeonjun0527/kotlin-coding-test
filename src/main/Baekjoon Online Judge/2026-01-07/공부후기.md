# Kotlin 핵심 요약 정리

---

## 1. 우선순위 큐(PriorityQueue)

### ✔ import
import java.util.PriorityQueue

### ✔ 기본 메서드 3개 암기
- add(value)     → 넣기
- peek()         → 꺼내지 않고 보기
- poll()         → 우선순위 가장 높은 값 꺼내기

### ✔ 내림차순 / 복합 정렬
- PriorityQueue<Int>(compareByDescending { it })
- PriorityQueue<Pair<Int, Int>>(compareByDescending<Pair<Int, Int>> { it.first }.thenBy { it.second })

---

## 2. 정수 나눗셈: 올림 / 반올림 / 버림

### ✔ 나눗셈 결과 기준
- 올림: (n + d - 1) / d
- 반올림: (n + d/2) / d
- 버림: n / d

---

## 3. size vs length

- **문자열만 length**
    - `"abc".length`

- **리스트/배열/컬렉션은 size**
    - list.size
    - array.size

---

## 4. 문자열 비교

- == → 내용 비교 (Java의 equals와 동일)
- === → 참조(주소) 비교

예:
"abc" == "abc"  → true  
"abc" === "abc" → 상황 따라 다름 (참조 비교니까)

---

## 5. 거듭제곱 (pow)

### ✔ Kotlin은 Int.pow 없음
→ Double로 해야 함

- import kotlin.math.pow
- 3의 8승: 3.0.pow(8)

(필요하면 결과를 Int로 변환: `3.0.pow(8).toInt()`)

---

## 6. 구간 개수

### ✔ 포함 구간 a..b 개수
- `maxOf(0, b - a + 1)`

### ✔ 미포함 구간 a until b 개수
- `maxOf(0, b - a)`

---

## 7. 기억할 핵심 문장 요약

- 우선순위 큐: **add / peek / poll**
- 정렬: **compareByDescending, thenBy**
- 문자열만 length, 나머지는 size
- == 는 내용 비교, === 는 주소 비교
- pow는 Double에서만: **3.0.pow(8)**
- 구간 개수:
    - 포함: b - a + 1
    - 미포함: b - a
