# DFS 냄새 찾기 연습

> 문제 설명만 보고 "이거 DFS다"를 떠올릴 수 있는가?

---

## 1. 적록색약 (BOJ 10026)

### <문제>
N×N 그리드에 R, G, B가 칠해져 있다. 같은 색이 상하좌우로 인접하면 같은 구역이다.
적록색약인 사람은 R과 G를 같은 색으로 본다.
적록색약인 사람과 아닌 사람이 각각 보는 구역의 수를 구하라.

### <정답>
**냄새:** `같은 색이 상하좌우로 인접하면 같은 구역` → **"연결된 덩어리를 세라"**

최단거리를 묻는 게 아니다. "같은 색으로 이어진 칸들을 하나의 덩어리로 묶고, 그 덩어리가 몇 개인지 세라"는 것이다.
한 칸에서 출발해 같은 색인 인접 칸을 죄다 방문하면 덩어리 1개가 완성된다. 이걸 모든 미방문 칸에 대해 반복하면 총 덩어리 수가 나온다.
적록색약 버전은 R과 G를 같은 색으로 취급하고 똑같이 덩어리를 세면 된다.
**"연결된 덩어리 세기 = DFS"**

---

## 2. 안전 영역 (BOJ 2468)

### <문제>
N×N 격자에 각 칸의 높이가 주어진다.
비가 오면 일정 높이 이하인 지점은 모두 잠긴다.
잠기지 않은 지점들이 상하좌우로 연결되면 하나의 안전 영역이다.
비의 양을 변화시킬 때, 안전 영역의 최대 개수를 구하라.

### <정답>
**냄새:** `잠기지 않은 지점들이 상하좌우로 연결 → 하나의 안전 영역` → **"연결된 덩어리를 세라"**

높이 기준값을 0부터 최대높이까지 바꿔가며, 매 기준마다 "잠기지 않은 칸들의 연결 덩어리"를 센다.
"기준 이상인 칸들만 남겨놓고 → 인접한 것끼리 묶어서 덩어리 개수 세기"를 반복하는 구조다.
덩어리 세기 자체는 전형적인 DFS다. 바깥에 브루트포스(기준값 순회)가 감싸고 있을 뿐이다.
**"조건을 만족하는 칸들의 연결 덩어리 세기 = DFS"**

---

## 3. 영역 구하기 (BOJ 2583)

### <문제>
M×N 모눈종이 위에 K개의 직사각형을 그린다.
직사각형 내부를 제외한 나머지 부분이 몇 개의 영역으로 나뉘는지,
각 영역의 넓이는 얼마인지 구하라.

### <정답>
**냄새:** `직사각형 외부의 나머지 부분이 몇 개의 영역으로 나뉘는지` → **"빈 칸의 연결 덩어리를 세라"**

직사각형을 격자에 표시하면 "벽"이 된다. 벽이 아닌 빈 칸들이 상하좌우로 이어진 것이 하나의 영역이다.
한 빈 칸에서 DFS로 연결된 빈 칸을 전부 방문하면 영역 1개가 완성되고, 방문한 칸 수가 곧 넓이다.
**"빈 칸의 연결 덩어리 세기 + 크기 구하기 = DFS"**

## 4. 촌수계산 (BOJ 2644)

### <문제>
부모-자식 관계가 트리로 주어진다. 부모↔자식은 1촌이다.
두 사람의 번호가 주어질 때 촌수를 구하라.
친척이 아니면 -1을 출력한다.

### <정답>
**냄새:** `부모-자식 관계 = 트리` + `두 사람 사이의 거리` → **"트리에서 두 노드 사이의 경로 탐색"**

트리는 사이클이 없으므로 두 노드 사이의 경로는 단 하나뿐이다.
한 사람(노드)에서 출발해 부모·자식 간선을 따라 DFS로 타고 가다가 상대방을 만나면, 거쳐온 간선 수가 곧 촌수다.
BFS로도 풀 수 있지만, 트리에서 경로 탐색은 DFS가 자연스럽다. 한 방향으로 쭉 타고 들어갔다가, 아니면 돌아오는 백트래킹 구조가 트리 탐색에 딱 맞기 때문이다.
**"트리에서 두 노드 사이 경로 찾기 = DFS"**

## 5. 트리 (BOJ 1068)

### <문제>
각 노드의 부모 정보로 트리가 주어진다 (루트의 부모는 -1).
노드 하나를 지우면 그 노드와 모든 자손이 제거된다.
지울 노드가 주어질 때, 남은 트리의 리프 노드 개수를 구하라.

### <정답>
**냄새:** `트리` + `노드 제거 후 리프 노드 개수` → **"트리를 타고 내려가며 전부 훑기"**

지울 노드의 서브트리 전체를 제거한 뒤, 남은 트리에서 자식이 없는 노드(리프)를 세야 한다.
루트에서 출발해 자식→자식으로 쭉 타고 내려가되, 지울 노드를 만나면 그 방향은 건너뛴다.
끝까지 내려갔는데 자식이 없으면(또는 유일한 자식이 지워진 노드였으면) 리프 +1.
이 "루트→리프까지 타고 내려가며 전부 훑기"가 DFS의 전형적 패턴이다.
**"트리를 전부 순회하며 조건에 맞는 노드 세기 = DFS"**

## 정리: DFS 냄새 체크리스트

| 냄새 | 핵심 질문 |
|------|----------|
| 연결된 덩어리 세기 | "인접한 것끼리 묶으면 몇 덩어리?" |
| 덩어리 크기 구하기 | "연결된 것을 싹 훑으면 몇 칸?" |
| 트리 순회/탐색 | "루트에서 타고 내려가며 전부 훑기" |
| 경로 존재 여부 | "A에서 B로 갈 수 있는가?" |
| **공통점** | **최단이 아니고, 연결된 것을 전부 탐색** |
